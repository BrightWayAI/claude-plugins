# BrightWay AI — Claude Plugin Marketplace

Cowork-compatible plugin marketplace for Claude Code.

## Available Plugins

| Plugin | Description |
|--------|-------------|
| **claude-cortex** | Always-on learning system — Claude gets smarter about you with every conversation. Passively observes preferences, captures knowledge, and adapts. |
| **bizdev-outreach** | Research a contact and draft personalized business development outreach. |
| **weekly-alignment** | Weekly cross-team alignment scanner — monitors Slack channels for overlapping work, conflicting decisions, and coordination gaps. |
| **lead-engine** | Turn your LinkedIn network into a warm-lead engine. Catches buying signals, warms prospects via comments and connection requests, drafts DMs in your voice, runs a 3-touch cadence, and generates pre-call briefs. |

## Installation

```
/plugin marketplace add BrightWayAI/claude-plugins
/plugin install claude-cortex@claude-plugins
/plugin install bizdev-outreach@claude-plugins
/plugin install weekly-alignment@claude-plugins
/plugin install lead-engine@claude-plugins
```
